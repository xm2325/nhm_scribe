## Decision

Do not expand to eval50 yet.

The next experiment should add a small OCR ensemble for Hespi `number`, `barcode`, and `database_label` crops using upscaling, contrast normalization, character whitelisting, and several single-line Tesseract modes. Handwritten collector and date fields should be handled in a separate HTR experiment rather than mixed into the catalogue-number change.
